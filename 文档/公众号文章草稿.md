# 零基础用 AI 搓出一个“童年遗憾弥补器”：我的全 AI 驱动独立游戏开发实录

大家好！今天想跟大家分享一个我最近利用纯 AI 制作、部署并开源的小游戏——**《人生模拟》**（英文名：*Game Life* / 时光回溯沙盒）。

这是一个支持 PWA（渐进式网页应用）的 AI 驱动 RPG 游戏。在游戏里，玩家可以回到“小学三年级放学后”，面对各种童年遗憾做出全新的选择，而 AI 扮演的世界引擎和 NPC 会根据你的选择，实时分裂出全新的平行宇宙分支，并绘制出一棵专属于你的“蝴蝶效应人生轨迹树”。

最不可思议的是：**这个项目从构思、UI 设计、编码、CORS 代理解决、PWA 配置、容灾存档，到最终一键部署至 Cloudflare，全过程 100% 由 AI 协助完成，我几乎只充当了“创意主导者与架构审核员”。**

以下是我的全流程开发记录，希望对想用 AI 做独立项目的朋友有所启发！

---

## 一、 简介：我们做了一个怎样的游戏？

你是否有过后悔的事情？如果当年放学后，你没有跟那个女孩擦肩而过；如果当年你鼓起勇气带死党一起走……人生会不会走向完全不同的方向？

这就是**《人生模拟》**的核心立意：**“童年遗憾弥补器” + “多重人生蝴蝶效应模拟”**。

### 核心玩法与特色：
1. **时空漩涡倒流**：点击历史节点，页面触发灰色复古滤镜与倒带旋转特效，带你“逆转时空”重新抉择。
2. **AI 动态剧情引擎**：后台大模型根据当前玩家属性（如：好感度、金钱、智力等）和前序剧情，实时生成 3 个推荐选项，同时支持**自定义文本输入**（你可以输入任何天马行空的行动），AI 会顺着你的行动续写剧情，拥有无限的自由度。
3. **蝴蝶效应分支树 (HTML5 Canvas)**：随着游戏的进行，Canvas 会动态绘制一棵发散的人生成长分支树。每一个选择都是一个新的分叉，所有分支清晰可见，并支持滚轮缩放、拖拽平移与点击回溯。
4. **动态 NPC 与拟真数字人**：
   - 游戏内置了三位初始 NPC：**妈妈（林素琴）、同桌（苏清婉）、发小（张小胖）**。
   - 他们的立绘是纯 SVG 绘制的，会根据当前对话的情绪（温柔、生气、惊讶、害羞等）自动切换表情，并配合 CSS 呼吸动效。
   - 接入了浏览器内置的 **Web Speech TTS 语音引擎**，实现声画同步的台词朗读。
5. **双轨存档防丢失**：自动存盘至 LocalStorage，并在每次重要抉择或导入时，自动生成灾备存档，即便遇到网络断开或解析崩溃，也能自动回滚，确保游玩进度安全。

## 二、 制作过程：四步把创意变成现实

抛弃了传统的“查文档、写 Bug、配环境”的痛苦流程，我这次把所有的繁琐工作都交给了不同的 AI 模型。整个制作过程被精简为以下极其丝滑的四步：

### 1. 视觉起步：用 ChatGPT DALL-E (image2) 一键搞定高品质 Logo 与图片资产
一个优秀的游戏必须在第一眼就抓住玩家的眼球。我使用 ChatGPT 里的图片生成功能，以“时光回溯与成长”为主题，输入了我的核心创意。
没过几秒钟，AI 就为我精准绘制出了一个**“怀旧游戏手柄”与“发芽成长大树”相结合的高清 Logo**，象征着通过游戏里的选择让人生重新发芽。我直接将这套高水准的视觉资产用作了游戏的 PWA（渐进式 Web 应用）桌面图标和主界面图标，免去了自己画图标或找设计素材的烦恼。

### 2. 结对编程：用 Codex + Gemini 3.5 flash 极速生成前端与交互代码
写代码是整个开发中工作量最大的部分。这一次我引入了 AI 协同编程工作流，搭配 Codex 与 Gemini 3.5 flash 双大脑：
- **UI 与界面逻辑**：Gemini 3.5 flash 展现出了极其强大的前端构建速度。我在提出“米色纸张背景”、“复古像素感”以及“蝴蝶效应发散人生轨迹树”后，它在几秒钟内便写出了美观的响应式页面结构，并且通过复杂的 Canvas 绘图算法，帮我实现了轨迹树的拖拽平移和鼠标滚轮缩放，整个交互过程丝滑无比。
- **系统联调与排错**：Codex 在全局上下文的理解上极为敏锐。每当我在多文件合并、本地测试遇到诸如接口超时或路径解析错误时，AI 都能瞬间识别报错日志并完成自动热修复，整个过程完全不需要我自己手写复杂的 Debug 逻辑。

### 3. Serverless 部署：用 Cloudflare 进行部署与接口代理
游戏做好了，如何发布给全球玩家，并且免除繁重的服务器运维？我选择了 Cloudflare 平台：
- **静态托管**：使用 Cloudflare Pages 一键连接 GitHub 仓库，代码每次更新推送，云端自动重构部署，玩家可以毫秒级加载游戏。
- **边缘接口代理 (Pages Functions)**：为了让玩家不用配置任何反向代理，AI 指引我使用 Cloudflare Pages Functions，在云端边缘节点上部署了微型的 Serverless 代理路由。它能够安全地替前端转发请求，不仅在源头上彻底解决了跨域 CORS 问题，还实现了“免输入 API Key 的即开即玩体验通道”，安全隔离了后端的密钥，真正做到了免维护部署。

### 4. 灵魂注入：用 Gemini 2.5 flash 模型作为底层世界引擎进行实时剧情驱动
游戏中的 NPC 和剧情是如何活起来的？我选用 Gemini 2.5 flash 作为核心驱动力。
- **动态世界引擎**：大模型就像一个不知疲倦的“幕后导演”。它实时读取玩家的各项属性（如好感度、名望等）以及之前的剧情节点，每次当玩家做出选择（或者在自定义框中输入天马行空的行动）时，它都会瞬间做出反应，流畅地续写故事。
- **NPC 情感与语音**：不仅如此，Gemini 2.5 flash 会在对话时同步分析并输出 NPC 的情感状态（如高兴、愤怒、惊讶等）。前台的 SVG 眨眼立绘会据此立刻切换表情，再配合浏览器内置的 TTS 语音，把一个活生生的、有感情的童年死党或母亲呈现在玩家面前。

## 三、 成本与体验：个人开发者的新时代

在这个全 AI 协同的项目里，成本和效率低得令人发指：

### 1. 财务成本：接近 ⚡️ 0 元！
- **服务器费用**：`0 元`。托管在 Cloudflare Pages，享受 CF 每日免费额度，个人及开源分享完全够用。
- **开发工具费用**：`0 元` / 仅 IDE/大模型 API 的基础调用成本（大部分甚至在免费额度内）。
- **图片生成费用**：包含在 ChatGPT 订阅中。

### 2. 研发效率：⚡️ 10倍速提升！
以往一个人写这样的游戏，需要画原型、写 CSS 布局、折腾 Canvas 缩放矩阵数学计算、写后端代理服务、配置 HTTPS 和部署。通常需要 **2 周** 以上的业余时间。
*而在这次的加持下，所有的文件交互、打包排错、代码重构均在几小时内完成。遇到 Bug 时，AI 可以在几秒钟内给出精准定位并一键热修复。*

### 3. 开发体验：从“码农”到“制作人”的转变
使用 AI 进行多文件级编码时，我真正感受到了**“结对编程”**的魅力。我提出“我想要个防丢的双轨存档”，AI 就会写出安全容灾闭环；我提出“加载有点单调”，AI 就写出了带有绿橘扫描动画的流光卡片。
开发者不再被细节的语法和 API 查阅所折磨，而是将 90% 的精力放在了**游戏机制设计、用户体验优化和创意打磨**上。
