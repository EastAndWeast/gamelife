# 零基础用 AI 搓出一个“童年遗憾弥补器”：我的全 AI 驱动独立游戏开发实录

大家好！今天想跟大家分享一个我最近利用纯 AI 制作、部署并开源的小游戏——**《人生模拟》**（英文名：*Game Life* / 时光回溯沙盒）。

这是一个支持 PWA（渐进式网页应用）的 AI 驱动 RPG 游戏。在游戏里，玩家可以回到“小学三年级放学后”，面对各种童年遗憾做出全新的选择，而 AI 扮演的世界引擎和 NPC 会根据你的选择，实时分裂出全新的平行宇宙分支，并绘制出一棵专属于你的“蝴蝶效应人生轨迹树”。

最不可思议的是：**这个项目从构思、UI 设计、编码、CORS 代理解决、PWA 配置、容灾存档，到最终一键部署至 Cloudflare，全过程 100% 由 AI 协助完成，我几乎只充当了“创意主导者与架构审核员”。**

以下是我的全流程开发记录，希望对想用 AI 做独立项目的朋友有所启发！

---

## 一、 简介：我们做了一个怎样的游戏？

你是否有过后悔的事情？如果当年放学后，你没有跟那个女孩擦肩而过；如果当年你鼓起勇气带死党一起走……人生会不会走向完全不同的方向？

这就是**《人生模拟》**的核心立意：**“童年遗憾弥补器” + “多重人生蝴蝶效应模拟”**。

### 核心玩法与特色：
1. **时空漩涡倒流**：点击历史节点，页面触发灰色复古滤镜与倒带旋转特效，带你“逆转时空”重新抉择。
2. **AI 动态剧情引擎**：后台大模型根据当前玩家属性（如：好感度、金钱、智力等）和前序剧情，实时生成 3 个推荐选项，同时支持**自定义文本输入**（你可以输入任何天马行空的行动），AI 会顺着你的行动续写剧情，拥有无限的自由度。
3. **蝴蝶效应分支树 (HTML5 Canvas)**：随着游戏的进行，Canvas 会动态绘制一棵发散的人生成长分支树。每一个选择都是一个新的分叉，所有分支清晰可见，并支持滚轮缩放、拖拽平移与点击回溯。
4. **动态 NPC 与拟真数字人**：
   - 游戏内置了三位初始 NPC：**妈妈（林素琴）、同桌（苏清婉）、发小（张小胖）**。
   - 他们的立绘是纯 SVG 绘制的，会根据当前对话的情绪（温柔、生气、惊讶、害羞等）自动切换表情，并配合 CSS 呼吸动效。
   - 接入了浏览器内置的 **Web Speech TTS 语音引擎**，实现声画同步的台词朗读。
5. **双轨存档防丢失**：自动存盘至 LocalStorage，并在每次重要抉择或导入时，自动生成灾备存档，即便遇到网络断开或解析崩溃，也能自动回滚，确保游玩进度安全。

## 二、 制作过程：全 AI 协同工作流与工具箱

在这个项目中，AI 不仅是一个“写代码的助手”，更是“设计师”、“架构师”和“运维工程师”。

### 🛠️ 我们的全 AI 开发工具链：

| 角色 | 采用工具 / 模型 | 具体职责与体验 |
| :--- | :--- | :--- |
| **作图/设计** | **ChatGPT: DALL-E 3** | 绘制高品质、复古游戏手柄与发芽大树结合的 PWA 专属 Logo 图标。 |
| **编码环境** | **Codex + Antigravity IDE** | 深度结对编程环境，支持多文件上下文感知、自动错误热重载、一键命令行打包。 |
| **逻辑/架构大脑**| **ChatGPT 5.5** | 负责核心架构设计（如双轨存档灾备、CORS 代理路由逻辑、Prompt 提示词模板设计与代码审查）。 |
| **前端/边缘开发**| **Gemini 2.5/3.5 Flash** | 负责极速的前端单页 HTML/CSS/JS 编写、Canvas 分支树布局算法以及 Cloudflare Functions 后端 Worker 的实现。 |
| **部署托管** | **Cloudflare Pages** | 提供零门槛的静态资产部署与 Serverless 函数托管，自动打通大模型反向代理。 |

### 🔄 开发演进过程：

#### 第一阶段：极简单页与 UI 视觉建立 (CSS 变量)
我们抛弃了沉重的框架，采用原生的 **HTML5 + Vanilla JS + Vanilla CSS**。AI 在 `style.css` 中建立了一套温润怀旧的色彩系统（米色纸张背景、绿橘流光交互），并编写了极其丝滑的打字机特效和 NPC 呼吸动画。

#### 第二阶段：核心状态机与 Canvas 分支树算法
如何展现“蝴蝶效应”？AI 建议用 HTML5 Canvas 绘制一颗对称发散的树。由于逻辑复杂，AI 快速写出了 `butterflyTree.js`，通过对各级子节点的相对坐标动态计算，实现了丝滑的滚轮缩放（Zoom）和拖拽平移（Pan），并且能够监听点击事件以定位最邻近节点进行回溯。

#### 第三阶段：突破浏览器跨域限制 (CORS 代理双通道)
直接在前端 Fetch 外部大模型 API 会遭遇浏览器的 CORS 跨域安全拦截。AI 为我设计了**“双通道代理”**：
- **开发环境**：在 `vite.config.js` 中拦截本地 `/api-proxy` 并实现动态主机转发。
- **线上环境**：在项目根目录新建 `/functions/api-proxy/[[path]].js`，利用 Cloudflare Pages Functions，在边缘服务器上建立同源代理，无感解除了跨域限制，且不依赖任何第三方代理服务器。

#### 第四阶段：PWA 改造与免配置体验通道
1. **应用名与图标更新**：用户提出将 PWA 名字改写为《人生模拟》，AI 自动同步修改了 `manifest.webmanifest` 和 HTML 的 apple-meta 属性，并完成了手柄 Logo 资产的打包覆盖。同时通过升级 Service Worker 的缓存版本（`game-life-shell-v5`），实现了客户端的无缝自动缓存更新。
2. **免配置“傻瓜”通道 (Gemini)**：为了降低开源后的游玩门槛（普通用户可能没有 API Key），AI 在前端设计了“免配置体验通道”，并新增了后台代理 `/functions/api-game.js`。此时，用户的 API 密钥被安全地存放在 Cloudflare 的后台环境变量中，前端不需要输入任何 Key 即可通过同源函数安全调用 Gemini 完成剧情推演，完美做到了“防 Key 泄露”与“即开即玩”。

## 三、 成本与体验：个人开发者的新时代

在这个全 AI 协同的项目里，成本和效率低得令人发指：

### 1. 财务成本：接近 ⚡️ 0 元！
- **服务器费用**：`0 元`。托管在 Cloudflare Pages，享受 CF 每日免费额度，个人及开源分享完全够用。
- **开发工具费用**：`0 元` / 仅 IDE/大模型 API 的基础调用成本（大部分甚至在免费额度内）。
- **图片生成费用**：包含在 ChatGPT 订阅中。

### 2. 研发效率：⚡️ 10倍速提升！
以往一个人写这样的游戏，需要画原型、写 CSS 布局、折腾 Canvas 缩放矩阵数学计算、写后端代理服务、配置 HTTPS 和部署。通常需要 **2 周** 以上的业余时间。
而这次，在 Antigravity + Gemini/GPT 大脑的加持下，所有的文件交互、打包排错、代码重构均在几小时内完成。遇到 Bug 时（例如由于转义符反斜杠导致的模板字符串渲染字面量问题，以及代理中 OPTIONS 请求无 Target 导致 DNS 挂掉的 502 报错），AI 可以在几秒钟内给出精准定位并一键热修复。

### 3. 开发体验：从“码农”到“制作人”的转变
使用 AI 进行多文件级编码时，我真正感受到了**“结对编程”**的魅力。我提出“我想要个防丢的双轨存档”，AI 就会写出 `preserveCurrentSaveAsBackup` 的安全容灾闭环；我提出“加载有点单调”，AI 就写出了带有绿橘扫描动画的流光卡片。
开发者不再被细节的语法和 API 查阅所折磨，而是将 90% 的精力放在了**游戏机制设计、用户体验优化和创意打磨**上。
