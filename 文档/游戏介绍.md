# 时光回溯沙盒：AI平行人生 RPG 游戏介绍

“如果人生可以重来，你会做出不一样的选择吗？”

《时光回溯沙盒：AI平行人生》是一款由大语言模型（LLM）驱动的**沉浸式文字成长模拟 RPG 游戏**。游戏将带你重温童年时光那些令人后悔、惋惜或感伤的关键节点，允许你通过改写当初的抉择，以可视化“蝴蝶效应分支树”的形式开启截然不同的平行人生，探索命运的万千走向。

---

## 🌟 核心特色与玩法

### 1. 回到童年，弥补那些年的遗憾
游戏默认从一个极具怀旧感的场景开始——**“小学三年级放学后”**。你将在破旧但温馨的平房里，面对唠叨但爱你的母亲、成绩优异的傲娇同桌、调皮但仗义的发小。那些当年让你后悔的争吵、没能说出口的话、弄丢的钢笔，都将在游戏里重新上演，等待你的全新回答。

### 2. 蝴蝶效应：可视化的“平行人生分支树”
在游戏界面右侧，常驻一棵由 Canvas 动态绘制的**“人生轨迹树”**。
* **时空标记**：你走过的每一步、经历的每一段剧情、做出的每一个抉择，都会被永久记录为树上的一个“时空节点”。
* **服下“后悔药”**：当你对某个选择的后果感到不满，或者好奇“如果当时我选了另一条路会怎样”，你可以随时在右侧树状图上点击对应的历史节点，一键回溯时光。游戏将在此处分裂出新的平行宇宙，开启别样的人生走向。

### 3. AI 驱动的动态世界与高自由度交互
游戏放弃了传统的固定剧本，通过接入先进的大语言模型，实现完全动态的剧情演变：
* **半开放抉择**：每次剧情推演完毕，AI 会根据现状提供 3 个符合逻辑的推荐选项。
* **自由键盘输入**：如果你对推荐选项不满意，可以通过底部的**“我想这样做...”**输入框，输入任何天马行空的行为（例如：“把手中的玻璃球送给苏清婉并向她道歉，然后叫上小胖去帮妈妈买酱油”），AI 将会立刻识别并实时衍生出契合的后续剧情。

### 4. 情感丰富的动态 NPC 社交网
你并不是孤独地在这个沙盒中探索。游戏内置了 3 位个性鲜明的核心 NPC：
* **林素琴 (母亲)**：望子成龙、嘴硬心软、严厉但内心深沉地爱你。
* **张小胖 (发小)**：有些懦弱、贪吃但为人仗义，总是无条件做你的跟屁虫。
* **苏清婉 (同桌)**：性格内敛、成绩优异、偶尔有些小傲娇，对你有种朦胧的情愫。

每个 NPC 都有独立的**好感度**、**信任度**，以及独特的**长期记忆库**。他们在对话时会呈现**呼吸、眨眼等微动作**，并根据情绪在平静、高兴、生气、难过、惊讶等 5 种矢量立绘表情间自然切换。配合浏览器内置的 Web Speech TTS 语音播报，实现声画同步的“数字人”交互。

### 5. 零门槛分享：双轨制存档系统
* **无缝自动存盘**：游戏进程全程自动保存在玩家浏览器的 LocalStorage 中，断电或刷新页面再次进入时秒级还原。
* **存档文件分发**：支持一键将当前的“多重人生分支树”导出为 `.json` 文件。你可以把存档发给朋友，他们导入后就能在你的分支树上继续探索，极具社区传播性。

### 6. 开源友好与隐私保护
* **密钥本地化**：为方便开源分发，游戏内置了“AI 多渠道配置面板”。你的 API Key 完全存储在本地浏览器，直连大模型或中转服务，绝不上传至任何游戏服务器，确保隐私与数据安全。
* **双通道免跨域代理**：内置 Cloudflare Pages Functions 代理通道，发布即可自动消除浏览器 CORS 跨域请求拦截，彻底解决 API 连通问题。

## 🛠️ 游戏运行与部署

* **前端技术栈**：HTML5 + Javascript (ES6) + Vanilla CSS + Vite
* **构建要求**：
  * 框架预设：`Vite`
  * 构建指令：`npm run build`
  * 输出目录：`dist`
* **托管平台**：完美适配部署在 **Cloudflare Pages**。部署后平台会自动识别根目录下的 `/functions` 并启动代理 Worker，免除所有反代跨域配置烦恼。
